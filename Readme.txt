Cuando lo ejecuten les aparecera el mensaje de como deben activarlo, luego de ahi podran darle los comados

tienen varios comandos a disposicon como:

"Abre" con este comando seguido del nombre de alguna aplicacion podran abrir lo que sea que el les pida sin restricciones
"Busca en google" con este comando podran pedirle que busque lo que sea en google, viene por defecto con la safesearch desactivado
"reproduce" Con este comando podran pedirle una cancion o video de youtube para que los lleve a youtube y reproduzca el video automaticamente
"Hora" con este comando podran estar al tanto de la hora sin necesidad de revisar el reloj o su telefono en caso de no tenerlo cerca
"Apagar" este comando apagara al asistente y ya no escuchara ningun comando a partir del momento en que este comando sea ejecutado

"Esta version ya cuenta con trabajo en segundo plano asi que no tendras que estar dentro del asistente para pedirle que haga cosas
